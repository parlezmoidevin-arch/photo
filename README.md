# Photos – parlezmoidevin.fr

Dépôt de stockage des photos pour le site parlezmoidevin.fr.

## Structure

| Dossier | Contenu |
|---|---|
| `Bouteilles/` | Photos de bouteilles de vin |
| `Propriétaires/` | Photos des propriétaires de domaines |
| `Domaines/` | Photos des domaines viticoles |
| `Parcelles/` | Photos des parcelles et vignes |

## Générer un lien raw

Pour chaque photo, le lien raw suit ce format :
```
https://raw.githubusercontent.com/parlezmoidevin-arch/photo/main/[Dossier]/[nom-fichier]
```

**Exemple :**
```
https://raw.githubusercontent.com/parlezmoidevin-arch/photo/main/Bouteilles/chateau-margaux.jpg
```
